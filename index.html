Formato decant

La forma más simple de descubrir tu próxima fragancia.

Un decant es una muestra en atomizador tomada desde un perfume original. Es ideal para probar aromas, llevarlos en el bolso, regalar o armar una colección personal.

3ml

Perfecto para probar una fragancia antes de decidir.

5ml

El formato equilibrado para uso semanal y rotación.

10ml

Para favoritos, viajes o perfumes de uso frecuente.
